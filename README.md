# README
This's my images application.

# Setup
Make sure you have Ruby and Rails installed.

* Ruby version: 2.5.3

Run in your terminal
```
git clone git@github.com:Maddalenap/images.git

cd images

bundle install

rails db:migrate
```
And run with

```
rails server
```

You can also use for debugging
```
rails console 
rails routes
```
